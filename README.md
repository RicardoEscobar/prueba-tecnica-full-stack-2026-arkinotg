# Cómo correrlo + decisiones + reflexión

## Cómo correrlo

Para correr el proyecto, primero debes clonar el repositorio y luego instalar las dependencias necesarias. Puedes hacerlo con los siguientes comandos:

```bash
python -m venv venv
source venv/Scripts/activate  # En Linux/MacOS se usa `source venv/bin/activate`
pip install -r requirements.txt
```

Una vez que hayas instalado las dependencias, puedes ejecutar el proyecto con:

```bash
python main.py
```
